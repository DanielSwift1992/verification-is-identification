# Task Board

@Metadata {
    @PageImage(purpose: card, source: "card-board", alt: "Task board")
}

A board laid out the way the dashboard is: columns of cards, each an org type. Every assignee links into the directory.

![Task Board](board)

Each task is one lane: its card at one station, dashed empty slots at the other two. Press a slot and the card stands there, dashed, a try. Any number of moves, there and back. Solid only ever appears at the built status, and the system holds it until a typealias changes and the build re-proves it.
